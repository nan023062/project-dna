外置 Agent 使用方式
1. 在目标项目安装 Cursor/Codex 工具后，IDE 会写入 MCP 配置并连接到本地 Client。
2. Agent 开始任务前先调用 get_context 获取模块边界，再调用 recall 拉取历史决策与约束。
3. 任务中关键决策和完成结果通过 remember 写回知识库，形成团队可复用的长期知识。
